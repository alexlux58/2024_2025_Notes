Command-Line Interface (CLI):

Starting OSSEC:


/var/ossec/bin/ossec-control start
Stopping OSSEC:


/var/ossec/bin/ossec-control stop
Restarting OSSEC:


/var/ossec/bin/ossec-control restart
Managing Agents:


/var/ossec/bin/manage_agents
This command allows you to add or remove agents in a server-agent setup.
Log Monitoring:

Alert Logs:


tail -f /var/ossec/logs/alerts/alerts.log
This command displays real-time alerts generated by OSSEC.
OSSEC Logs:


tail -f /var/ossec/logs/ossec.log
This shows OSSEC's internal logs, useful for troubleshooting.
